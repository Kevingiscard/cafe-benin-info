# ☕ Café Bénin — Site Vitrine

Site web vitrine pour **Café Bénin**, un café authentique proposant les saveurs d'Afrique de l'Ouest.

## 🌐 Voir le site en ligne

👉 **[https://kevingiscard.github.io/cafe-benin-info](https://kevingiscard.github.io/cafe-benin-info)**

## 📋 Sections du site

- **Accueil** — Hero animé avec appel à l'action
- **À propos** — Histoire du café et statistiques clés
- **Menu** — Boissons, snacks, plats et desserts avec onglets interactifs
- **Galerie** — Présentation visuelle de l'ambiance
- **Témoignages** — Avis clients
- **Contact** — Informations, horaires et formulaire de contact

## 🛠️ Technologies utilisées

- HTML5 sémantique
- CSS3 (variables, animations, responsive design)
- JavaScript vanilla (sans dépendances)
- Google Fonts (Playfair Display + Lato)

## 🚀 Déploiement

Ce site est déployé via **GitHub Pages** sur la branche `main`.
